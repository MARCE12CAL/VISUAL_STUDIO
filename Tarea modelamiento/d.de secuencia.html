Diagrama de Secuencia

  1. Usuario envía solicitud de búsqueda de coches al Sistema.
  2. Sistema consulta la base de datos de Coches.
  3. Sistema devuelve la lista de coches al Usuario.
  4. Usuario selecciona un coche y envía solicitud de alquiler.
  5. Sistema solicita datos de alquiler y calcula el coste total.
  6. Usuario confirma el alquiler.
  7. Sistema guarda el alquiler en la tabla alquileres.
